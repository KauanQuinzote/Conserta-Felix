# Conserta Felix — Client

Projeto criado com Next.js, TypeScript e Tailwind CSS.

Como rodar:

1. Instalar dependências

```bash
npm install
```

2. Rodar ambiente de desenvolvimento

```bash
npm run dev
```

3. Build e start

```bash
npm run build
npm start
```

Observações:
- Arquitetura: app router do Next (pasta `app`).
- Tailwind está configurado em `tailwind.config.cjs`.
