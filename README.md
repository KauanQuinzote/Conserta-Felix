The system's name and interface are inspired by the hero **Fix-It Felix Jr.** from the *Wreck-It Ralph* animated movie. ### System Scope

The system is divided into two main panels, reflecting the different actors and their responsibilities:

1.  **Workshop Panel (Administration):**
    * Complete management of clients, vehicles, and orders (Service Orders).
    * Definition of service types offered (Body Repair, Polishing, Maintenance, etc.).
    * Control of vehicle types the workshop services (Motorcycle, Car, Van).
    * Profile update and security settings.

2.  **Client Panel:**
    * Viewing and editing personal profile and registered vehicles.
    * Real-time tracking of order/service status.
    * Quick scheduling of new services.
